# Injection Writeups

This folder contains my writeups for legal injection labs completed on OWASP Juice Shop.

The goal is not only to document final payloads, but to explain:

- what I observed
- how I built my hypothesis
- what mistakes I made
- why the vulnerability worked
- what the real-world impact would be
- how to remediate it

## Covered patterns

- Generic login SQL injection bypass
- Targeted login SQL injection bypass
- SQLite schema extraction
- Soft-deleted product bypass
- API and business logic abuse after SQL injection
