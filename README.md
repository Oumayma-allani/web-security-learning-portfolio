# web-security-learning-portfolio
This repository documents my practical learning path in web application security.

The goal is not to publish payload collections or copied theory, but to document how I study vulnerabilities, reproduce them in controlled lab environments, understand their impact, and learn how they can be prevented.

# Purpose

This portfolio is used to track my progress in web security through:

concept notes written in my own words,
lab writeups from legal training environments,
testing methodologies,
common mistakes I encountered,
remediation notes and defensive lessons.

# Topics Covered
Authentication
